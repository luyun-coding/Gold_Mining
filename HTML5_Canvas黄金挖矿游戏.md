# HTML5 Canvas黄金挖矿游戏

## Core Features

- 钩子摆动(0.8Hz)

- 发射/碰撞/回收(450px)，速度4，初始长度40px

- 难度配置与黄金分值映射

- UI去重（HUD由DOM显示）

- 倒计时按整秒递减并实时更新

- 动态目标分：基于当前得分×难度系数

- 炸弹固定减少10秒时间
